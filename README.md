## 9 Sınıflı Görüntü Sınıflandırma İçin Özelleştirilmiş CNN Mimarisi

Bu proje, Bilgisayar Mühendisliği lisans eğitimi kapsamında Tasarım Projesi dersi için geliştirilmiştir. Hazır modeller (transfer learning) kullanılmadan, katman yapısı ve parametreleri sıfırdan kurgulanmış özgün bir Evrişimli Sinir Ağı (Convolutional Neural Network - CNN) mimarisi tasarlanmıştır.

Proje kapsamında 9 farklı sınıfı içeren bir veri seti üzerinde çalışılmış ve geliştirilen model ile %60 doğrulama (validation) başarısı elde edilmiştir. Bu çalışma, derin öğrenme temellerini, katman optimizasyonunu ve model başarısını etkileyen faktörleri analiz etme amacı taşımaktadır.

## Custom CNN Architecture for Multi-Class Image Classification

Description:
Developed within the scope of the Computer Engineering Design Project course, this repository contains a deep learning model built to classify images across 9 distinct categories.

Instead of utilizing pre-trained models, a custom Convolutional Neural Network (CNN) architecture was designed and implemented from scratch to understand the intricacies of feature extraction and layer optimization. The model achieved a validation accuracy of 60%, demonstrating effective learning capability on the provided dataset.

# Key Highlights:

Custom Architecture: A layer-by-layer designed network focusing on optimizing computational efficiency and accuracy.

Performance: Achieved 60% accuracy on a 9-class dataset.

Core Concepts: Implements fundamental deep learning concepts including convolution, pooling, and dropout for regularization.
